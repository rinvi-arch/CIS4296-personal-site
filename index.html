Projects

Some projects that I loved working on.

In this project I've worked in a group to design an intern storing and tracking system for a non-profit organization called Mitzvah Circle. We're designing this system which provide historical interns more efficiently by having all their information in one place, allowing interns to update their information: contacts, education, occupation/work title. It will also enable historical interns to request a letter of recommendation. Executive Director will automatically be notified via email with the necessary information (name, purpose, when they last worked). The purpose of this project was to help this organization to get grand money for them to be able to pay these interns.
